v3 notes

W10-W18: all-black TYP0 — 0 を mint から ink black に変更。brand monogram (0 + dots = mint) と visually separate な役割になる: monogram = mint mark / wordmark = black name。色で機能を分ける design system。

★ W10 (Lalezar) と W11 (Jost Black) = 確定 monogram と font 一致するので brand 統合感が最強。一番素直な選択。

Brand system preview: 5 通り combo 並べた:

  • combo 1: M13 + W10 + T08 (Lalezar 統一 + Inconsolata 時刻)
  • combo 2: M13 + W10 + T11 (Lalezar 統一 + Bebas Neue station)
  • combo 3: M14 + W11 + T08 (Jost 統一 + Inconsolata)
  • combo 4: M14 + W11 + T11 (Jost 統一 + Bebas Neue)
  • combo 5: M13 monogram + W11 wordmark + T08 (混合: chunky 0 + sans wordmark)

designer 観察:

  • Lalezar 統一 (combo 1, 2) = CPFM / Brain Dead 寄り、P4 subculture punch
  • Jost 統一 (combo 3, 4) = geometric sans modernist、editorial quiet luxe 寄り
  • 混合 (combo 5) = brand mark (chunky) と wordmark (clean) を contrast、Margiela / ALD 系の "logo system" 流儀
  • T08 Inconsolata = terminal 系、4:81 が「コードの時刻表記」感、cult 寄り
  • T11 Bebas Neue = vintage station / scoreboard 系、4:81 が「物理的な時刻表示」感、loud

提案: combo 1 (Lalezar 統一 + Inconsolata) か combo 5 (混合) が brand 強度高い。tig 好み次第。
