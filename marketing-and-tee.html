v5 vs v5-alt 比較

  • v5 (上): Front = TYP0 stet wordmark (center) / Back = 0+dots (lower-left)
    → conventional brand convention、Stüssy / Awake NY 流儀。「これは TYP0stet」と即読める
  • v5-alt (下): Front = 0+dots monogram (center) / Back = TYP0 stet (lower-left)
    → Margiela 0 line 流儀 subversion。「謎の 0 mark」→ 歩き去ると名前判明、cult 強度高

v5 = 商業的に正解、v5-alt = brand 強度高。Vol.1 launch では 両方作って drop ごとに切り替える選択肢もあり (Wave 01 = v5 / Wave 02 = v5-alt 等)。

使い方

  • Marketing assets: 各 canvas を browser で右クリック → 「画像として保存」 OR スクリーンショット → SNS にそのままアップ可
  • 解像度: 現在 preview は scaled-down、final 投稿時は Canva で SVG を import → 高解像度 export (1080×1080 / 1200×630 etc)
  • Tee mockup: あくまで CSS 配置 sketch、実 mockup は AI 画像生成 (Midjourney 等) or 印刷 sample で確認
  • IG grid: 9 tile を順番 (tile 1 → 9) で post すると profile grid が完成。3×3 既に埋まる
  • Story テンプレ: 上 wordmark、中 quote、下 monogram の 3 段構成が brand-coherent
