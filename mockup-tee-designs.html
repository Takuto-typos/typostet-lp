v4 観察ポイント

Mint × mint stet (*): 同系色は invisibility 寄り、deeper mint + hairline ink outline で救済。tonal は luxe sign だがほぼ読めない trade-off。

White / Black の punch: white と black base は stet 3 色全部が cleanly 立つ、最も "wear it loud" 寄り。P4 subculture target には black + red の punch が刺さる可能性大。

Orange の地位: red より warm、ink との contrast 強い。Bone + orange は editorial 寄り、Black + orange は subculture punch、Olive + orange は autumnal。red より一段 sophisticated に感じる可能性。

Bone + red: tig 元 Canva 案の DNA、最も "校正者の赤ペン" semantic を保つ。

Back の universal mint: 全 shirt 共通の back signature。Black/Olive base に mint 0 + dots = 強く立つ。White/Bone は subtle で adult。

裾 4:81: 全 shirt mint 固定。tonal な Mint base ではほぼ invisible、ink outline で救済済。
