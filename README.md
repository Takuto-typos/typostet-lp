# typostet-lp

TYP0stet (タイポステット) brand LP.

- **brand thesis**: 失敗できる、を誇りに。
- **domain**: typostet.com (Cloudflare Registrar)
- **hosting**: Cloudflare Pages (auto deploy from `main`)
- **redirect**: typos.jp → typostet.com (301)

## development

```bash
# preview locally
python3 -m http.server 8080
# open http://localhost:8080
```

## deploy

git push to `main` → Cloudflare Pages auto deploys to typostet.com.
